# NOT PIXEL BOT | Auto Claim and Print Pixel

## Install NOT PIXELBOT
```
git clone https://github.com/zamzasalim/notpixel.git
```
```
cd notpixel
```
```
pip install -r requirements.txt
```
## Token User | Token Query | Update Token
**Use WebTelegram - Inspect Bot NotPixel - Application - __telegram__initParams - Copy Query= | user=**

## Paste Token User
```
nano config.py
```
**After Paste CTRL + XY**

## RUN Bot
**Create Screen**
```
screen -S notpixel
```
**Run Bot**
```
python3 run.py
```
## Troubleshoot Screen 
**Close Screen | CTRL + AD**

**Go back to Screen**
```
Screen -r notpixel
```
**Delete Screen**
```
screen -S notpixel -X kill
```
